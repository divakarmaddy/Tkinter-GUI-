Welcome to my repository! This is a collection of Python GUI applications I developed using the Tkinter library. Each project is designed to showcase various functionalities, ranging from basic applications to more advanced tools.

Below are the projects included in this repository:

ATM SIMULATOR: A basic GUI-based ATM simulation to perform operations like withdrawal, deposit, and balance inquiry.

ROCK PAPER SCISSORS: A fun interactive game where the user can play against the computer.

AUTHENTICATION SYSTEM: A login and registration system to demonstrate authentication mechanisms.

AUTOMATIC SIGNAL SYSTEM: A traffic signal simulation with automatic light transitions.

MANUAL SIGNAL SYSTEM: A traffic signal system where the user can control light transitions manually.

CALCULATOR: A simple calculator with basic arithmetic operations.
